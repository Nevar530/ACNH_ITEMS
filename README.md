# ACNH Items Database

## Features
- Search by name, notes, or recipes
- Filter by tag
- DIY-only toggle
- Sort by price / profit
- Works on mobile
- Add to Home Screen support (PWA-ready)

---

## How to Use on Phone
1. Open the link above in your browser.
2. Tap **Share**.
3. Select **Add to Home Screen**.
4. It now works like an app.

---

## Data Structure
Each item contains:

- `item`
- `raw_value`
- `price`
- `profit`
- `margin`
- `tag`
- `diy`
- `notes`
- `recipes`
